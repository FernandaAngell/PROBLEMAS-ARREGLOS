# 🧠 Problemas con Arreglos y Matrices en Python

## 📚 Descripción

Este repositorio contiene ejercicios desarrollados en Python 🐍 enfocados en el uso de arreglos (listas) y matrices, aplicados a problemas prácticos.

Se trabajan conceptos como:

* Listas y matrices
* Ciclos (for, while)
* Validaciones
* Operaciones matemáticas
* Análisis de datos

---

## 📂 Contenido

### 🍦 Problema 8 - Control de helados

Sistema para gestionar compras de helados con control de presupuesto, cantidades y estadísticas.

### 🔢 Problema 9 - Matriz cuadrada

Ingreso de una matriz con números de 4 cifras, cálculo de mayor, menor, suma de pares y verificación de número capicúa.

### 👨‍💼 Problema 10 - Productividad empleados

Uso de matrices para calcular tiempos de trabajo y análisis por empleado y por día.

### 🍕 Sistema pizzería

Simulación de pedidos y cálculo de costos.

### 🏥 Sistema triage

Clasificación de pacientes según nivel de urgencia y tiempos de atención.
